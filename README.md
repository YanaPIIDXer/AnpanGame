*cocos2d-x 3.17*

クローン→ビルドまでの作業

※cocos2d-xのインストールが完了している前提での作業となります。  
1.適当なディレクトリを作成し、そこにクローン。  
2.以下のコマンドでプロジェクトを作成。  
  *cocos new AnpanGame -p com.yanap.anpangame -l cpp -d {作業用ディレクトリ}*  
3.cocosコマンドで生成したディレクトリ（作業用ディレクトリ）から以下のディレクトリを削除。  
・Classes  
・Resources  
4.１でクローンしておいたファイルをゴッソリ作業用ディレクトリにコピペ。  
5.proj.win32\AnpanGame.slnを開き、ビルドできることを確認。

※１：WebAPIを使用したゲームですが、サーバ側（ＡＰＩ）は公開していません。  
※２：Windows環境でのみ対応しています。Macには対応していません。（プロジェクト構成とか。）
